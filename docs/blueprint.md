# **App Name**: Digital Alchemist

## Core Features:

- Hero Section Display: Display a hero section that matches the look and feel of the reference image, including typography, image placeholder, and layout.
- AI-Powered Style Advisor: Suggest additional UI elements based on a generative AI tool. Suggest placements, contents and styling for Work, Play, About, Resume/Contact in navigation elements. Use the information from the 'About' section of the page, to ensure all content is styled in a similar manner.

## Style Guidelines:

- Primary color: #2515D8 to convey intelligence, sophistication, and the high-tech arena.
- Background color: White for a clean and modern look.
- Text color: Black for optimal readability and contrast.
- Headline font: 'Belleza', a stylish sans-serif suited to artistic or high-end contexts, to reinforce the portfolio's unique personal branding. Body font: 'Alegreya' for legibility if longer text is used elsewhere in the portfolio.
- Mimic the hero section layout from the reference image, focusing on a full-width design with prominent typography. Ensure the layout is responsive and adapts to different screen sizes.
- Use minimalistic icons that complement the tech and design aspects of the portfolio, reflecting both DevOps and UI/UX design skills. Keep icons simple and visually consistent.
- Incorporate subtle animations on hover states for interactive elements, adding a layer of polish to the user experience without being distracting.
